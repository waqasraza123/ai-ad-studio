# AI Ad Studio

AI Ad Studio is a premium portfolio product for generating product ad concepts and rendering short vertical ads.

## Workspace

- `apps/web` Next.js application
- `apps/worker` background worker
- `packages/shared` shared domain contracts and types
- `packages/config` runtime configuration utilities
- `packages/ui` reusable UI primitives
- `packages/providers` provider contracts and adapters
- `packages/media` media pipeline utilities

## Phase 1 goals

- monorepo scaffold
- tooling and CI
- working web app
- working worker app
- shared package boundaries
- green lint, typecheck, test, build
