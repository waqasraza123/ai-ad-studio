# Phase 26

Comparative review UI for batch outputs, with side-by-side previews, decision notes, and winner selection per batch.

## Acceptance criteria

- render_batches support winner export selection and decision notes
- dedicated batch review page exists
- batch review page shows side-by-side previews for all batch exports
- one export can be selected as the winner for a batch
- decision note is persisted on the batch
- winner selection writes a notification and job trace for auditability
