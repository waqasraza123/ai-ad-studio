# Phase 21

Reusable template system for branded ad styles, scene packs, and preset CTA end cards.

## Acceptance criteria

- durable ad_templates table exists
- projects can select a template
- template scene packs shape scene planning during render
- template CTA presets shape the final end-card rendering
- project page shows a template selector and current template preview
- render metadata, usage events, and traces persist template information
