# Phase 23

Brand kit expansion with saved logos, palettes, typography tokens, and automatic template-theme binding per project.

## Acceptance criteria

- durable brand_kits table exists
- projects can select a brand kit
- brand kits store palette and typography tokens
- project render flow binds selected brand kit into template planning and final CTA styling
- project page shows brand kit selector
- settings page allows editing default brand kit tokens
- render metadata, usage events, and traces persist brand kit information
